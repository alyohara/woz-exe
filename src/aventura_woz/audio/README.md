# Audio — WOZ.exe

Temas chiptune se generan solos la primera vez (`.wav` en esta carpeta).

## Reemplazar con tu música

Poné archivos con estos nombres (prioridad: `.ogg` > `.mp3` > `.wav`):

| Archivo | Cuándo suena |
|---|---|
| `theme_boot` | Splash / boot / perfil |
| `theme_act1` | Homebrew, stack, módem, portal |
| `theme_act2` | Neon, cantina, maze, antecámara |
| `theme_act3` | Bahía, trie, núcleo HAL |
| `theme_ending` | Epílogo |
| `theme_gameover` | Sin créditos |

Ejemplo: `theme_act1.ogg` pisa el chiptune generado.

Volumen: `config.MUSIC_VOLUME`. In-game: tecla **M** mute.
